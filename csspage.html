CSS Cheat Sheet Page

Connecting an external style sheet to a html file:

<head> …
	<link href=”folder/file_name.css”>
</head>

CSS properties I most used

color
background-color
background-image: url(“fileName”);
background-repeat
font-family (use double quotes when the value contains spaces)
font-size
text-align
line-height
text-decoration
align-items
justify-content
width
height
margin (-top/bottom/right/left)
padding (-top/right/bottom/left)
border (-top/right/bottom/left)
border-radius
cursor
position
float:left (moves an element to the left until it hits another element)
clear:both (removes floats and the element will be placed below the one before)

ID Reference

To apply a style to a specific element use # + the id of this element:

#idName{
properties;
}

CLASS Reference

To apply a style to a class of elements use . + the class given:

.className{
properties;
}

CHILD Reference

Use > to access the child selector or :nth-child to access a specific child:

ul > li{
properties;
}

li:nth-child(2){
properties;
}

Links States

a:link (unvisited link)
a:visited (visited link)
a:hover (moving the mouse over the link)
a:active (a link when clicked)

Flexbox Layout

Use the property display:flex; for the parent container

Other flexbox properties for the parent:

flex-wrap
flex-direction
align-items
justify-content
align-content

Flexbox properties for the children:

order
align-self

Grid Layout

Use the property display:grid; for the parent container

Other grid properties for the parent:

justify-content
align-content
grid-template-columns
grid-template-rows

Grid properties for the children:

grid-column
grid-row
grid-area
