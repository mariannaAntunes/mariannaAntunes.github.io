HTML Cheat Sheet Page

The basic structure of a HTML page

<!DOCTYPE html>
<html>
  <head>
	 <title></title>   ->   title bar of the web browser
	</head>
	<body>
	</body>
</html>

Unpaired Tags

Some tags do not have a closing tag (unpaired tags). Some that I used the most were:

<p>
<br>
<input>
<img>
<link>
<hr>


*If there is an element that I know I will have to style individually, I can use the attribute “id” to give this element a specific name.
*If there is a group of elements that I know I will have to give the same style, I can use the attribute “class” to give this group a specific name.
*To add comments use: <!--  …  -->

Special Characters

Special characters usually have a specific code:

space = &nbsp;
< = &lt;
> = &gt;


*Link to find more examples: Common HTML entities used for typography - W3C Wiki

Unordered Lists / Ordered Lists

<ul> / <ol>
  <li>…</li>
	<li>…</li>
	<li>…</li>
</ul> / </ol>

Links

*Tag: <a>Text to be clicked on</a>
*Attribute: href=”put the link here or the file name for a page of your own website”
*The attribute target=”_blank” will open the external link in a new tab
*To link a section in the page, give an id to this section and put this id in the href=”#idgiven”

Images

*Tag: <img>
*Attribute: src=”folder/file_name”

Tables

<table>
  <tr>   ->    create a row
	 <th>Content goes here</th>   ->    for the table head
	 <th>Content goes here</th>
  </tr>

  <tr>
    <td>Content goes here</td>   ->    create a cell
		<td>Content goes here</td>
  </tr>
</table>

*Use the attribute colspan=”number of columns” to merge columns
*Use the attribute rowspan=”number of rows” to merge rows
