Strings

Most used string methods:

stringName.indexOf(“searched string”)
stringName.lastIndexOf(“searched string”)
stringName.replace(“string that will be replaced”, “string that will replace”)
stringName.search(“searched string”)
stringName.slice(inicial index, final index)
stringName.split(character where the string will be split in more than one string stored in a array of strings)
stringName.toLowerCase()
stringName.toUpperCase()

*length property: stringName.length
*To concatenate strings use + and +=
*To access a individual character of a string use stringName.charAt(index)

Random Numbers

To generate random numbers use:

let low = minNumber;
let high = maxNumber;
console.log(Math.floor(Math.random() * (high - low)) + low);

Array

To declare an array use: let arrayName = [];

Most used array methods:

arrayName.push(new element to be added to the end of the array)
arrayName.unshift(new element to be added to the beginning of the array)
arrayName.pop() – removes the last element of the array and return it
arrayName.shift() - removes the first element of the array and return it
arrayName.slice(starting index, final index excluded) – returns the extracted subarray and does not modify the original array
arrayName.concat(addedArrayName)
arrayName.indexOf(searched array) – use the same for lastIndexOf()
arrayName.sort() – Modifies the content of the original array and is used to put elements in alphabetical order
arrayName.sort(function(a,b){return a-b;}); - sorting an array of numbers
